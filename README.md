# Debian Configuration Lab

Repositori ini merupakan lingkungan pembelajaran untuk mempelajari dan
mempraktikkan konfigurasi server Linux menggunakan **Debian**.

## Tujuan
- Mempelajari konfigurasi Debian dari dasar hingga lanjutan
- Melatih kemampuan administrasi server dalam lingkungan lab
- Persiapan untuk kompetisi, sertifikasi, maupun penerapan di dunia kerja

## Materi yang Dipelajari
- Konfigurasi jaringan
- Manajemen user dan permission
- Firewall (nftables)
- DNS, Web Server, dan VPN
- Manajemen service dan troubleshooting sistem

## Lingkungan
- Sistem Operasi: Debian Linux
- Virtualisasi: VMware / VirtualBox
- Client: Windows / Linux

## Penggunaan
Lab ini dibuat khusus untuk **belajar dan latihan**.
Silakan bereksperimen dan mencoba konfigurasi secara langsung.
